# D.eSystem_alpha-6.0.6

D.eSystem 6.0.6 alpha is the newest alpha release of the D.eSystem operating system.
This version introduces the first graphical user interface (GUI) in the history of D.eSystem and marks a major milestone in the transition from text‑based to fully graphical OS design.

# D.eSystem 6.0.6 alpha download:
  https://github.com/D-electronics-scratch/D.eSystem-6.0.6-alpha-iso-download

# alpha 6.0.6 features
- First GUI on D.eSystem

- Graphical calculator (first GUI application)

- Major stability improvements across the kernel and UI modules

- Improved framebuffer handling

- More reliable hardware initialization

# System requirements
D.eSystem 6.0.6 alpha runs on:

Any 64‑bit virtual machine (QEMU, VirtualBox, VMware)

Real hardware with:

64‑bit CPU

UEFI firmware

Basic framebuffer support

# This release continues the evolution of D.eSystem into a modern, UEFI‑based, bare‑metal operating system with real hardware support and a growing graphical subsystem.
